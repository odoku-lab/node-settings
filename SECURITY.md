# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in @odoku-lab/settings, please report it by emailing the maintainers at **odoku-lab@googlegroups.com**.

Please do **not** open a public GitHub issue for security vulnerabilities.

We will acknowledge receipt within 48 hours and provide an estimated timeline for a fix.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ Active          |
| < 0.1   | ❌ Not supported   |

## Disclosure Policy

- We will investigate and fix the issue as quickly as possible
- A security advisory will be published on GitHub once the fix is released
- We will credit the reporter unless they request anonymity
